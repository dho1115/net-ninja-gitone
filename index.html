How to configure your username & password in github:

git config --global user.name [your username]

git config --global user.email [your email]

To retrieve the user name:

git config user.name

How to see which files are staged & which are committed:

git status

How to add a file to the commit:

git add [filename] or git add .

Observation about .gitignore (removing files from staging)..

So, let's say I just added something to my staging area that I did not want to. What I did was, I just went into .gitignore, inserted that file (e.g. package-lock.json) and then that file was automatically removed from staging!!!

Another way to remove files from staging.

git rm --cached filename.html (or .js or whatever extension).

What returns is rm 'filename' and you will see that file inside Changes if you click the icon on the left.

If you make changes to a file after you git add the file:

Just do another git add [changed file].

Committing a file:

git commit -m " ... ". The m stands for message.

If you see something like 75 insertions after you did git commit -m, that means there are 89 lines of new code changes.

To see all your different commits:

git log.
