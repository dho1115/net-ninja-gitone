How to configure your username & password in github:

git config --global user.name [your username]

git config --global user.email [your email]

To retrieve the user name:

git config user.name

How to see which files are staged & which are committed:

git status

How to add a file to the commit:

git add [filename] or git add .

Observation about .gitignore (removing files from staging)..

So, let's say I just added something to my staging area that I did not want to. What I did was, I just went into .gitignore, inserted that file (e.g. package-lock.json) and then that file was automatically removed from staging!!!

Another way to remove files from staging.

git rm --cached filename.html (or .js or whatever extension).

What returns is rm 'filename' and you will see that file inside Changes if you click the icon on the left.

If you make changes to a file after you git add the file:

Just do another git add [changed file].

Committing a file:

git commit -m " ... ". The m stands for message.

If you see something like 75 insertions after you did git commit -m, that means there are 75 lines of new code changes.

To see all your different commits:

git log.

Type q to quit git-log.

To see the abbreviated version of your commit:

git log --oneline

There are THREEdifferent types of commits:
Checkout commit, revert commit and reset commit.

Checkout commit.

  1. STEP 1: git log --oneline to see the commits and the commit id for each commit.
  2. STEP 2: git checkout [the id of the commit you want to checkout].
  3. STEP 3: After you checked out that branch (this branch is read only), you do git checkout master to return back to the master branch because what happens is when you checkout a branch id using git checkout [id], you go away from the master branch to whatever the branch id is. You are no longer on the master branch, so what you need to do is, you must do git checkout master to return back to the master branch.

git revert

git revert _branchID

You MAY see a screen with a bunch of ~. To get out of that, you just hold down the shift [shift] + [:] (shift - colon) and then type "wq" to get out of that.

When you do git log --oneline again, you will see Revert "...".

How to create a new git branch.

TWO WAYS:

  1. git branch [new branch name].

    To checkout that new branch:

    git checkout [new branch name].

  2. git checkout -b feature-a.

git branch [new branch name].

To checkout that new branch:

git checkout [new branch name].

How to DELETE a branch:

git branch -D [branch]

NOTE: lower 'd' only works if the branch is merged.

Git reset

git reset [id] --hard.

How to MERGE a branch.

git merge [branch you want to merge with].
